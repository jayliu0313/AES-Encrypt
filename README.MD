# AES_Encrypt
這是一款可以加密任何文件，並且解密為原來的AES加密系統，其中包含了ECB、CBC、CFB、OFB、CTR等區塊加密來提升我們安全。  
直接編譯執行，照著裡面的步驟選擇要加密還是解密、使用的鑰使長度、區塊加密的方式、個人密碼。
